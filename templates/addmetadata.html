Add Metadata To Public Data


To add new metadata to your public data at GNPS, you'll need to create a metadata spreadsheet describing each individual mass spectrometry file. This will need to fit into our own metadata template that is entered according to a controlled vocabulary. Check out this template.

Once your have create your own metadata file saved as a tab separated text file based on the template, you can check if the format is valid below with a drag and drop.

Once your metadata has been updated save your file as gnps_metadata.tsv exactly and update your dataset with this gnps_metadata.tsv in the other file category. This metadata will be imported automatically each evening.
